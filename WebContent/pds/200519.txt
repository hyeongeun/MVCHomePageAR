1. InputStream : 바이트기반 입력 스트림의 최상위 클래스로 추상 클래스 이다.
- FileInputStream, BufferedInputStream, DataInputStream

1-1) read() 메소드 : 입력 스트림으로부터 1바이트 읽고 바이트 리턴.

2. OutputStream : 바이트 기반 출력 스트림의 최상위 클래스로 추상 클래스 이다.
- FileOutputStream, PrintStream, BufferedOutputStream, DataOutputStream

3. BufferedInputStream/BufferedOutputStream : 일정한 단위로 묶어서 입출력하는 클래스이다. 파일의 크기가 큰 경우 읽어오는 시간을 줄일 수 있다.

4. 스레드 : 하나의 응용 프로그램에서 실행되는 작업 단위?.

5. 멀티스레드 : 하나의 프로그램에서 다수의 작업이 가능한 것. 멀티 프로세서는 다수의 프로그램을 사용해서 많은 작업을 하는 것.
